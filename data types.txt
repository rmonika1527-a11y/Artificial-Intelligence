Numeric Types

int (Integer) → whole numbers

a = 10

float → 
decimal numbers

b = 3.14

complex → real + imaginary numbers

c = 2 + 3j
 2. Sequence Types

Store multiple values in order.

list → mutable (can change)

my_list = [1, 2, 3, "apple"]

tuple → immutable (cannot change)

my_tuple = (10, 20, 30)

string (str) → text data

name = "Riya"

 3. Set Types

Unordered collection, no duplicates.

set

my_set = {1, 2, 3, 3}
# Output: {1, 2, 3}
 4. Mapping Type

Store data in key-value pairs.

dict (dictionary)

student = {
    "name": "Monika",
    "age": 20,
    "course": "BSc CS"
}
5. Boolean Type

Used for True/False values.

is_active = True
 6. Binary Types

Used for raw binary data.

bytes

x = b"hello"

bytearray

y = bytearray(5)

memoryview

z = memoryview(b"abc")
